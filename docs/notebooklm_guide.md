# NotebookLM 使用指南

> 将 132 篇论文导入 Google NotebookLM，让 AI 帮你快速生成领域综述。

## 📥 导入步骤

1. 访问 [Google NotebookLM](https://notebooklm.google.com/)
2. 创建新笔记本
3. 上传 `pdfs/` 文件夹中的 132 篇 PDF 文件
4. 使用下方提示词进行提问

## 💬 推荐提示词

### 一、总体概览

```
请根据我上传的所有 ICLR 2026 论文，生成一份关于 "人工智能辅助药物发现 (AIDD)" 领域的综述。

请首先提供一个总体概览，包括：
1. 这批论文涵盖了哪些主要研究方向？
2. 今年有哪些突破性进展或新兴趋势？
3. 哪些技术方法应用最为广泛？

请用中文回答，并引用具体论文标题作为支撑。
```

### 二、分主题深度分析

```
请针对以下每个主题进行深度分析：

1. 蛋白质结构预测与设计
   - 有哪些论文专注于这一方向？
   - 核心创新点是什么？

2. 小分子生成与优化
   - 使用了哪些生成模型？
   - 如何解决可合成性问题？

3. 分子对接与相互作用预测
   - 这些方法如何超越传统物理模拟？

4. 分子动力学与构象采样
   - 使用了哪些加速技术？

请为每个主题列出 3-5 篇代表性论文，并总结其核心贡献。
```

### 三、技术方法对比

```
请对这些论文中使用的技术方法进行横向对比：

1. 扩散模型 (Diffusion Models)
   - 有多少论文使用？在哪些任务上表现最好？

2. 流匹配 (Flow Matching)
   - 与扩散模型相比有何优势？

3. 图神经网络 (GNN)
   - 有哪些新的 GNN 架构被提出？

4. 大语言模型 (LLM)
   - LLM 在化学/生物领域的应用情况？

请用表格形式总结各方法的代表论文和优缺点。
```

### 四、数据集与基准

```
请总结这些论文中提到的数据集和基准测试：

1. 有哪些新的基准数据集被提出？
2. 常用的评估指标有哪些？
3. 哪些方法在标准基准上取得了 SOTA？

请列出主要数据集的名称、规模和代表论文。
```

### 五、未来研究方向

```
基于这些论文，请分析 AIDD 领域的未来研究方向：

1. 目前仍存在哪些未解决的挑战？
2. 哪些新兴方向值得关注？
3. 对于刚进入该领域的研究者，你有什么建议？

请结合具体论文进行分析。
```

### 六、生成完整综述报告

```
请将上述所有分析整合成一份完整的综述报告，采用以下结构：

# ICLR 2026 AIDD 领域综述

## 1. 引言
## 2. 研究主题分类
   - 2.1 蛋白质结构预测与设计
   - 2.2 小分子生成与优化
   - 2.3 分子对接与相互作用预测
   - 2.4 分子动力学与构象采样
   - 2.5 药物发现应用
## 3. 核心技术方法
## 4. 数据集与基准
## 5. 趋势与展望
## 6. 结论

请确保内容详实、逻辑清晰，并引用具体论文。
```

## 💡 使用技巧

1. **分步提问**：按照上述顺序逐步提问，效果更好
2. **追问细节**：如果某个回答不够详细，可以追问
3. **请求引用**：可以说 "请列出支持这一观点的具体论文"
4. **生成表格**：对于对比分析，要求以表格形式呈现

## ⚠️ 注意事项

- NotebookLM 单次最多上传 50 个文件，需分批上传
- 上传后等待几分钟让系统完成索引
- 生成的综述建议人工审核后再使用
